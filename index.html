YT Unblocked Helper

Open the site, auto-fill a YouTube link, then copy the generated URL.

Step 1 — Install / use the Autofill bookmarklet

Drag this to your bookmarks bar, or right-click → “Bookmark link”. Then visit youtubeunblocked.live and click it.

Autofill & Go

When clicked on youtubeunblocked.live, it will prompt you for a YouTube URL, place it into the big field, and press Go!

Step 2 — Copy the final link

After the proxy opens the video page, click this bookmarklet to copy the current page URL. It will also send the link back to this helper so it can auto-copy it for you.

Copy Current URL

Works on most modern browsers; includes a fallback if clipboard permissions are blocked.

Step 3 — Collected proxy link

Keep this page open. When the bookmarklet sends the link back, it appears here and is copied for you.
